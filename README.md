# AIToDo

AIToDo 是一款基于 **PyQt5** 的桌面待办事项工具，支持任务管理、提醒通知和 AI 自然语言创建任务。

## 功能
- 添加、编辑、删除任务
- 标记任务完成/未完成
- 设置任务优先级
- 设置提醒时间
- 支持 AI 自然语言识别
- 支持深色模式和主题切换
- 支持系统托盘

## 说明
你可以直接输入待办内容，也可以使用 AI 输入自然语言，例如：

- `明天下午3点开会`
- `把学习Python设为高优先级`
- `删除已完成任务`

程序会自动解析并执行对应操作。

## 环境
- Python 3
- PyQt5
- requests

## 运行
```bash  
pip install pyqt5 requests  
python main.py
