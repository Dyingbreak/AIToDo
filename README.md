# AIToDo

AIToDo 是一款支持 AI 自然语言输入的桌面待办事项工具，可以把一句话自动转成待办操作。

## 功能
- 添加、编辑、删除任务
- 标记完成/未完成
- 设置优先级和提醒
- 支持 AI 智能解析
- 支持主题切换和深色模式
- 支持系统托盘

## AI 用例
你可以直接输入类似下面的话：

- `明天下午3点开会`
- `后天提醒我交作业`
- `把学习Python改成高优先级`
- `把写周报设为低优先级`
- `删除已完成任务`
- `清空所有任务`
- `完成第一个任务`
- `取消完成第一个任务`
- `给“项目汇报”设置明天上午10点提醒`
- `把“买菜”改成晚上8点提醒`
- `删除“测试”这个任务`

程序会自动识别这些指令并执行相应操作。
